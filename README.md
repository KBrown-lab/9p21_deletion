#  A high-penetrance intergenic variant at 9p21 confers melanoma susceptibility
This repository provides bioinformatics analysis scripts for the manuscript titled: "A high-penetrance intergenic variant at 9p21 confers melanoma susceptibility” .  
This study was conducted as part of the MelaNostrum Consortium: https://dceg.cancer.gov/research/cancer-types/melanoma/melanostrum   

The repository includes:  
    - Jupyter Notebook for running the deep learning model AlphaGenome and rendered HTML output  
    - Bash script to perform structural variant call on bam file using ClinSV software  
    - Fasta files containing human assembly hg38 sequences used in AlphaGenome prediction  
    - WGS data information (dbGaP accession number phs004649.v1)  

Software dependencies:  
    - Python 3.10  
    - Jupyter notebook or Jupyter Lab  
    - AlphaGenome (version 0.6.1) and all dependencies for Python packages are listed within the python notebook  
    - ClinSV 1.1  
    - Singularity 4.3.7  

<ins>Note</ins>: HTML file review is not available and needs to be downloaded and open locally on your computer
