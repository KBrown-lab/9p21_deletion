# This repo contains R and python scripts used in the 9p21 deletion manuscript
  ## Description:
       AlphaGenome_prediction_9p21deletion.ipynb: python notebook for using AlphaGenome to predict effects of the 9p21 deletion on gene expression and chromatin state, used in Fig3 and FigS5
       Fig2_loopingdata_analysis.R: R script used to analyze HiChIP peak-to-peak data (Fig2)
       Supplementary_Tables-Figures.R: R script used to generate some supplementary tables and figures
