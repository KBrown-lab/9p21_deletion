#  A high-penetrance intergenic variant at 9p21 confers melanoma susceptibility
This repository provides bioinformatics analysis scripts for the manuscript titled: "A high-penetrance intergenic variant at 9p21 confers melanoma susceptibility” .   
The repository includes:
    - Python Notebook for running the deep learning model AlphaGenome 
    - Bash script to perform structural variant call on bam file using ClinSV software
    - Fasta files containing human assembly hg38 sequences used in AlphaGenome prediction

Software dependencies:
    - Python ≥ 3.8
    - Jupyter notebook or Jupyter Lab
    - All dependencies for Python packages are listed within the python notebook
