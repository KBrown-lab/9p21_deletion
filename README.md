#  9p21_deletion 
This repository provides bioinformatics analysis scripts for the manuscript titled: "A high-penetrance intergenic variant at 9p21 confers melanoma susceptibility” .  The study reported in this manuscript is a part of the MelaNostrum consortium (https://dceg.cancer.gov/research/cancer-types/melanoma/melanostrum).  
The repository contains the major scripts and data information for the analyses described in the manuscript.   
For detailed instructions and further explanations, please see the README file located within each folder.
